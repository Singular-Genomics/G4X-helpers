# G4X Viewer Installation

### Prerequisites
- Node.js (version 16 or higher)
- npm

### Installation Steps
1. Clone the G4X Viewer repository:
```bash
git clone https://github.com/Singular-Genomics/G4X-viewer.git
```

2. Navigate to the project directory:
```bash
cd G4X-viewer
```
3. Install the dependencies:
```bash
npm install
```
4. Start the application:
```bash
npm run dev
```

!!! note
    The application will now launch on your local server. The default Vite development server runs on http://localhost:5173. If this port is already in use, Vite will automatically select an available port and display it in the console.
